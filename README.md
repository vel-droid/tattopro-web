## Overview

TattooPro is a modern web application designed specifically for tattoo salon owners and masters to manage their business efficiently. The application includes client management, appointment scheduling, revenue tracking, and more.

### Tech Stack

**Frontend:**


- Next.js 16+ with React
- TypeScript
- Tailwind CSS for styling
- Next.js API Routes

**Backend:**

- Express.js
- Node.js with TypeScript
- JWT Authentication
- CORS enabled
